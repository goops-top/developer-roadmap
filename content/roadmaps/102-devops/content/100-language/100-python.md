# Python


Python 是一种多范式语言。作为一种解释语言，代码在编写时立即执行，并且 Python 语法允许以函数式、过程式或面向对象的编程方式编写代码。

Python 经常被推荐为新程序员应该学习的第一门语言，因为它注重可读性、一致性和易用性。

这也带来了一些缺点，因为该语言在大多数生产任务中性能不是特别好。
